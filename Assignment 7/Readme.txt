### HTML

WHAT IS HTML
HTML is the standard markup language for creating web pages.

WHY HTML IS USED
HTML is used to structure content on the web.

What is Emmet & List 5 Emmet
Emmet is a plugin for writing HTML/CSS faster; examples include `html:5`, `ul>li.item$*5`, `div#id.class`, `a[href=""]{Link}`, `form>input:text*2+input:submit`.

SHORT CUT CREATE of Emmet Boiler plate of HTML
The Emmet shortcut for an HTML boilerplate is `html:5`.

WHAT STRUCTURE OF HTML
An HTML document structure includes `<!DOCTYPE html>`, `<html>`, `<head>`, `<title>`, and `<body>`.

WHAT IS HEAD TAG
The `<head>` tag contains meta-information about the HTML document.

WHAT IS BODY TAG
The `<body>` tag contains the visible content of the HTML document.

IMPORTANT 3 ELEMENTS OR TAGS
Three important tags are `<div>`, `<a>`, and `<img>`.

WHAT IS DIFFERENT B/W ELEMENTS & TAGS IN HTML
Tags are HTML code like `<p>`, while elements include tags, content, and attributes like `<p>This is a paragraph.</p>`.
